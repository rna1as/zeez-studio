# ZEEZ Studio — Keycap Customizer

Versi awal aplikasi custom keycap/keychain untuk Android berbasis PWA.

## Fitur
- Pilih jumlah slot 1 sampai 6
- Auto slot berdasarkan teks
- Input nama/emoji
- Pilih warna case/body
- Pilih warna keycaps per slot
- Pilih warna tulisan
- Kode desain otomatis
- Simpan preview sebagai PNG
- Kirim format order ke WhatsApp

## Cara pakai cepat
1. Buka `index.html` di browser Android/Chrome.
2. Isi teks, warna, dan nomor WhatsApp toko.
3. Tekan **Simpan gambar** untuk menyimpan preview.
4. Tekan **Kirim WhatsApp** untuk membuka chat WhatsApp dengan format order otomatis.

## Cara dijadikan aplikasi di HP
Agar bisa di-install seperti aplikasi:
1. Upload folder ini ke hosting HTTPS, misalnya Netlify, Vercel, GitHub Pages, atau hosting pribadi.
2. Buka URL-nya di Chrome Android.
3. Pilih menu titik tiga → **Add to Home Screen / Install app**.

## Catatan
- Nomor WhatsApp disimpan lokal di perangkat pengguna.
- Palet warna dibuat mendekati warna produk custom keycap/keychain populer: putih, cream, pink, coral, merah, kuning, mint, teal, biru, lavender, ungu, navy, hitam, abu, clear.
- Ini adalah desain UI original untuk ZEEZ Studio, bukan salinan brand atau tampilan toko lain.
